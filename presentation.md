## Конверсионные доработки web-витрины

- **Главная:** стильные CTA-элементы (не обязательно кнопки — красивые ссылки тоже работают), переработка шапки (градиентная подложка + непрозрачность при наведении, без потери вау-эффекта), пагинация, заметная боковая гендерная навигация, товарные блоки, очевидный путь к футеру.
- **Посадочные коллекций:** CTA, перелинковка с коммерцией через Shop the Look, уникальный шаблон баннера — не дублировать главную, не дезориентировать пользователя.
- **Чекаут:** убрать обязательную регистрацию, авторизация через внешние сервисы (Sber ID — полный профиль без заполнения полей), формы под 152-ФЗ, сначала адрес / точка самовывоза, потом данные и оплата, оплата при получении для Click& Collect, BNPL-сервисы.
- **Карточка товара:** дополнить эмоциональный контент рациональной информацией — УТП, описание, UGC, блоки апсейла и кросс-сейла, информация о BNPL.

**Профит:**

- ✓ Рост конверсии в оформлении заказов на **20–45%**.
- ✓ Снижение показателя отказов на checkout.
- ✓ Рост среднего чека за счёт реко-блоков и Shop the Look.
- ✓ Снижение юридических рисков (152-ФЗ).

## Доставка на адрес и ПВЗ

- Сейчас — только Click& Collect с предоплатой. Цель — полноценная доставка.
- Собственные магазины всех 4 брендов как точки самовывоза.
- Курьерская доставка (собственная + агрегаторы).
- Сторонние ПВЗ (СДЭК, Boxberry и др.), Почта России.

- Внутренняя система бизнес-правил для выбора оптимального перевозчика — оптимизация костов.
- **Открытые вопросы:** отгрузка интернет-заказов с каких складов? Нужна ли отгрузка с розничных точек?

**Профит:**

- ✓ Рост онлайн-заказов в **2–3 раза**.
- ✓ Снятие главного барьера для новых клиентов.
- ✓ Покрытие всей географии России.
- ✓ Оптимизация логистических костов через бизнес-правила.

## SEO: аудит и базовая оптимизация

- Технический аудит: скорость, Core Web Vitals, краулинговый бюджет.
- Структура URL, каноникалы, hreflang.
- Микроразметка Schema.org для товаров, категорий, хлебных крошек.
- Оптимизация мета-тегов, заголовков, alt-текстов.
- Устранение дублей контента и битых ссылок.
- XML-sitemap, robots.txt, внутренняя перелинковка.

**Профит:**

- ✓ Фундамент для всего органического роста.
- ✓ Поисковая органика (особенно брендовая) — **самый конверсионный** трафик на сайт прямо сейчас.
- ✓ Без базовой технички остальные инвестиции в SEO бессмысленны.

## Доработки коммуникаций

- Починить траст почтового домена — перестать попадать в спам.
- Ревизия текущих цепочек, составление карты коммуникаций + предложения по развитию.
- Велком-цепочка: вознаграждение за заполнение профиля — обогащение данных + промокод = доп. продажи. Запуск после первого заказа или неполной регистрации.
- Триггеры: брошенная корзина, избранное, снижение цены, новые поступления (товары, похожие на просмотренные — интеграция с рекомендациями).
- Аудит и проработка шаблонов и стратегии дайджест-рассылок.
- Запуск push-коммуникаций после запуска мобильного приложения.

**Профит:**

- ✓ Рост клиентской базы.
- ✓ Качественные коммуникации вместо спама в никуда.
- ✓ Рост конверсии с email-канала.
- ✓ Возврат «спящих» клиентов.
- ✓ Обогащение профилей для персонализации.

## Омниканальная программа лояльности

- Единая ПЛ на ВСЕ бренды — онлайн и офлайн.
- Бонусы, кэшбэк, персональные предложения, уровни.
- Бренды **не конкурируют**, а дополняют друг друга — покупатель и члены его семьи могут открывать для себя другие бренды экосистемы, и наоборот.
- Единая карточка клиента для аналитики, сегментации и персональных коммуникаций.
- Альтернатива: 4 изолированных небольших интернет-магазина. Или единая экосистема, где каждый бренд усиливает остальные.
- **+20–30%** повторных покупок.

**Профит:**

- ✓ Рост клиентской базы.
- ✓ **+20–30%** повторных покупок.
- ✓ Кросс-продажи между брендами — рост LTV.
- ✓ Точечные персональные коммуникации с каждым клиентом.
- ✓ Данные для принятия бизнес-решений.
